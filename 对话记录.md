
### 3. 接入方式

#### Python
```python
import ollama
response = ollama.chat(model='gemma3:4b', messages=[{'role': 'user', 'content': '你好'}])
```

#### Node.js
```javascript
import ollama from 'ollama';
const res = await ollama.chat({ model: 'gemma3:4b', messages: [{ role: 'user', 'content': '你好' }] });
```

#### Java
```java
// 使用 HTTP Client 调用 REST API
POST http://localhost:11434/api/chat
```

## Ollama 功能
1. 运行本地 LLM
2. 代码生成/补全
3. 聊天对话
4. 文本生成
5. 代码解释
6. API 服务
7. 多模型管理

## API 端点
- 聊天：`POST /api/chat`
- 生成：`POST /api/generate`
- 拉取模型：`POST /api/pull`
- 列出模型：`GET /api/tags`
